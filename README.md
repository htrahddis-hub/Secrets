# Authentication-Secrets
The app is made using Node js, Express framework and MongoDB.

To run this app locally 
1.clone the repo to your PC.
2.install the required module to the todolist-app directory
```sh
$ npm install 
```
3.run the app
```sh
$ node app.js
```
